# Single Day Weather App

A simple React app that fetches and displays real-time weather data using the OpenWeatherMap API.

## Features

- Search by city name
- Displays:
  - City name
  - Current temperature
  - Weather description
  - Weather icon
- Handles loading and error states

## Built With

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- OpenWeatherMap API
